# Byobu / tmux

Prefix: **Ctrl+Space** or **F12**

Zellij uses the same bindings — see `cheatsheet zellij`.

## Tabs

| Key | Action |
|-----|--------|
| `prefix + c` | new tab |
| `prefix + n` | next tab |
| `prefix + p` | previous tab |

## Panes

| Key | Action |
|-----|--------|
| `prefix + s` | horizontal split |
| `prefix + v` | vertical split |
| `Ctrl+h/j/k/l` | move between panes (vim-aware) |
| `Ctrl+\` | toggle last pane |

## Resize (prefix + Alt)

| Key | Action |
|-----|--------|
| `Alt+h/j/k/l` | resize pane |
| `Alt+Up/Down` | resize up / down |

## Copy mode (vi)

| Key | Action |
|-----|--------|
| `v` | start selection |
| `y` / `Y` | yank and exit |
